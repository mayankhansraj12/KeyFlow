# KeyFlow support

## Supported candidate

KeyFlow `0.1.7` is a manually updated macOS beta. Support covers installation, permissions, keyboard app-launch shortcuts, fixed gesture features, the Sound Bar, screenshot behavior, the window switcher, configuration recovery, and privacy-redacted diagnostics on macOS 15 or later.

Raw multitouch is experimental until the physical matrix is complete. Automatic updates, Mac App Store distribution, arbitrary scripts, imports, and the wider BetterTouchTool-scale roadmap are outside this candidate’s support boundary.

## Requesting help

Use [GitHub Issues](https://github.com/mayankhansraj12/KeyFlow/issues) for non-sensitive bugs and support requests. Include:

- KeyFlow version/build and macOS version;
- Mac/trackpad class without serial numbers;
- exact steps and expected/observed behavior;
- the redacted diagnostics export when useful.

Do not post configuration files, typed text, action values, screenshots containing private content, signing material, or security vulnerabilities in a public issue. Security issues follow `SECURITY.md` and require private vulnerability reporting once the repository owner enables it.

Support has no guaranteed response-time commitment during beta. A release owner must define service targets and an independent contact method before a commercial/public production launch.
